Sirotez&Melangez Administrateur

Sirotez&Melanger Administrateur est une application de bureau qui permet de gérer les tables de la base de données de l'application Sirotez&Melanger (Appli de jeux de cartes, voir projet 2). Elle sert de CRUD pour les tables de la base de données, ce qui signifie qu'elle permet de créer, modifier, supprimer et bien sûr afficher les éléments des tables.

Lien github


Pour ce projet, j'ai besoin de votre participation. Je vais vous demander d'installer WampServer. Si on vous demande des informations personnelles, ne les renseignez pas et passez directement au téléchargement. Une fois l'installation faite, lancez Wamp afin de finaliser l'installation. En acceptant les licences, suivez les étapes jusqu'au bouton "Installer", puis cliquez dessus. Enfin, cliquez sur "Terminer". Pour lancer le logiciel, rendez-vous dans votre barre d'outils en bas à droite. Vous devriez voir un logo vert. Cliquez dessus et allez sur "localhost". Une fois la page web ouverte, cliquez sur PhpMyAdmin. Utilisez "root" comme nom d'utilisateur, ne mettez pas de mot de passe et choisissez le serveur MySQL. Une fois sur l'interface de PhpMyAdmin, importez le fichier SQL du répertoire "Sirotez&Melange2.0". Si vous préférez utiliser XAMPP ou un autre logiciel de ce type, vous pouvez le faire. Cependant, faites attention à ne pas avoir deux serveurs MySQL tournant sur les mêmes ports, sinon cela ne fonctionnera pas. Si mes explications ne sont pas assez claires, je vous prie de consulter le tutoriel ci-dessous./p>
